Para poder ejecutar el programa, debemos estar dentro de la carpeta correspondiente a la clase Main
    ...Tarea_S1_05\src\n1ejercicio1

Dentro de esta carpeta, ejecutamos el comando "javac" acompañado del nombre del archivo que contiene el "main".
En este caso es "Main.java", finalmente presionamos la tecla "enter".

    javac Main.java

Para poder utilizar el método para listar los archivos, ejecutamos el comando "java" acompañado del Main
 y finalmente presionamos la tecla "enter".

    Ejemplo:
    java Main

Te solicitará que le pases la ruta a listar para completar la operación.
