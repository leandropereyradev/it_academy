Para poder ejecutar el programa, debemos estar dentro de la carpeta correspondiente a la clase ListarAlfabeticamente
    ...Tarea_S1_05\src\n1ejercicio1

Dentro de esta carpeta, ejecutamos el comando "javac" acompañado del nombre del archivo que contiene el "main".
En este caso es "Main.java", finalmente precionamos la tecla "enter".

    javac Main.java

Para poder utilizar el método de la clase ListarAlfabeticamente para listar los archivos, ejecutamos el comando "java"
acompañado del Main y, como parámetro, la ruta a listar entre comillas, finalmente precionamos la tecla "enter".

    java Main "C:\Users\formacio\Desktop\Leandro Pereyra\it_academy"
